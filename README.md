# Work Permit & LAP Client Guide

This is a static website prepared for quick internal reference during client calls.

## Files
- `index.html` – the whole site
- `data.json` – structured reference data you can update later

## How to publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html` and `data.json` to the root of the repository.
3. In GitHub, open **Settings** → **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save.
7. GitHub will give you a public website link after deployment.
